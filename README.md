## 📖 專案介紹

這是一個展示自動化交易策略（EA - Expert Advisor）開發成果的網站。包含三個已完成開發並經過實測的交易策略，所有數據均來自真實的市場回測結果，為投資者提供透明、完整的策略評估資訊。

### 什麼是EA（自動化交易）？
EA是一種電腦程式，能夠根據預設的規則自動進行買賣交易，無需人工干預。就像設定好的機器人，24小時監控市場並在適當時機執行交易。

## 🎯 三大交易策略

### 📊 美股指數策略
**專注市場：** 美國科技股指數 (NASDAQ100)  
**策略特色：** 智能雙模式自動切換

- **智能判斷：** 每天開盤前自動分析市場狀況
- **雙模式運作：** 根據市場類型選擇最適合的交易方式
- **實測成績：** 6個月測試期間獲勝率達72.37%
- **投資報酬：** 1000美元本金獲利7,237美元

### 🥇 黃金雙軌策略  
**專注市場：** 國際黃金市場  
**策略特色：** 兩種策略同時運行

- **趨勢策略：** 當黃金走勢明確時，跟隨主要方向獲取較大利潤
- **震盪策略：** 當黃金在區間波動時，進行短線來回交易
- **智能風控：** 根據市場波動自動調整風險控制
- **適應性強：** 能適應各種不同的市場環境

### ⚡ 黃金突破策略
**專注市場：** 國際黃金市場  
**策略特色：** 專業突破交易系統

- **突破捕捉：** 專門抓住黃金價格突破重要關鍵點的機會
- **假突破過濾：** 使用先進技術避免落入假突破陷阱
- **延遲進場：** 不急於進場，等待市場確認突破有效性
- **實測成績：** 獲勝率60%，但盈利因子高達3.07
- **投資報酬：** 1000美元本金獲利1,548美元

## 💡 投資術語解釋

### 基礎概念
- **獲勝率（勝率）：** 在所有交易中賺錢交易的百分比
- **盈利因子：** 總獲利除以總虧損，大於1表示整體獲利
- **夏普比率：** 衡量投資風險與報酬平衡的指標
- **最大回撤：** 投資過程中最大的虧損幅度

### 策略術語
- **突破交易：** 價格突破重要支撐阻力位時的交易策略
- **趨勢跟隨：** 跟隨市場主要方向進行交易
- **區間震盪：** 在價格區間內進行來回交易
- **動態止損：** 根據市場波動自動調整的停損機制

### 技術指標
- **移動平均線：** 平滑價格波動，判斷趨勢方向
- **RSI指標：** 判斷市場超買超賣狀態
- **ATR指標：** 衡量市場波動程度
- **MACD指標：** 捕捉趨勢變化信號
